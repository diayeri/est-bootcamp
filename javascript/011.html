문자열 메소드

indexOf(), lastIndexOf()

- 입력값: "문자열", (시작인덱스)

includes()

- 입력값: "문자열"
- 결과값: t / f

match()

- 정규표현식과 함께 사용
- index 번호를 찾지는 않음
- 결과는 배열로 출력됨

substring()

: 문자열의 일부 가져오기 *slice와 비교!
- 입력값: 시작인덱스, 종료인덱스
- 종료인덱스가 없으면 끝까지 출력
- 음수, 숫자가 아닌 값은 0으로 인식함
- 시작인덱스가 종료인덱스보다 값이 큰 경우, 자동으로 치환하여 계산

* slice()

: 문자열의 일부 가져오기
- 입력값: 시작인덱스, 종료인덱스
- 음수, 숫자가 아닌 값은 0으로 인식함
- 시작인덱스가 종료인덱스보다 값이 큰 경우, 빈 문자열 반환 (substring과는 다르게, 자동치환 안함!)

substr()

(DEPRECATED: 웹표준에서 제거될 가능성 높음)
- 입력값: 시작인덱스, 종료인덱스
- 음수, 숫자가 아닌 값은 0으로 인식함
- 시작인덱스가 종료인덱스보다 값이 큰 경우, 빈 문자열 반환 (substring과는 다르게, 자동치환 안함!)
